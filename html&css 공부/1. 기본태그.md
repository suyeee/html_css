[시멘틱 태그]

  웹 페이지에서 중요한 영역을 나누는 덩어리 태그들



  header : 웹 문서의 머리말 영역

  nav : 웹 문서에서 링크 등의 메뉴 영역

  figure : 웹문서에서 동영상, 사진 등 다양한 멀티미디어를 담는 영역

  main : 웹 문서의 본문으로 콘텐츠를 담는 영역

  aside : 웹 문서의 본문과 연관성이 적은 외부 영역

  section : 웹 문서의 특정 영역을 그룹화

  article : 웹 문서에서 기사나 개별 콘텐츠를 담는 영역

  footer : 웹 문서의 꼬리말 영역
